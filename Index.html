About the Game

WHAT IS
VOID RUN?

You run. The void chases. Simple as that.


Void Run is a brutally fast endless runner built for players who want a real challenge. Dodge, jump, shoot, and survive through Neon City, Dark Zone, and the Void Realm — each section harder, faster, and more chaotic than the last.


At the end waits the Void Tyrant. Good luck.
